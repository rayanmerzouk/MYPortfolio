# Portfolio — Rayan Merzouk

Portfolio professionnel destiné à un recrutement en ingénierie logicielle / full-stack junior.

## Objectifs
- Présenter un profil Master 2 Génie Logiciel, sans exagération.
- Mettre en valeur des projets académiques avec une lecture technique claire.
- Fournir un frontend sobre, responsive et SEO-friendly.

## Stack
- React (Vite)
- Tailwind CSS v4
- TypeScript

## Structure
- `src/App.tsx` : page principale et sections
- `src/data/portfolio.ts` : contenu (profil, projets, compétences)
- `src/index.css` : thème, typographie, utilitaires

## Démarrage
```bash
npm install
npm run dev
```

## Personnalisation rapide
- Modifiez les sections et textes dans `src/data/portfolio.ts`.
- Renseignez email / LinkedIn / GitHub dans la section Contact.
- Ajoutez des captures locales si nécessaire.

## SEO
Les balises principales sont dans `index.html` (title, description, Open Graph).
